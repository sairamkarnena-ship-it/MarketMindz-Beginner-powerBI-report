# 📊 MarketMindz – Marketing Analytics Dashboard (Beginner Level Project)

## 📌 Project Overview

This project analyzes marketing and customer data for a retail food & beverage company using Power BI.

The objective of this project was to:
- Understand customer demographics
- Analyze product performance
- Evaluate marketing campaign results
- Identify purchase behavior patterns

This is a beginner-level BI project focused on data understanding, transformation, and visualization.

---

## 🗂 Dataset Information

- Total Customers: 2,240
- Total Columns: 28
- Domain: Retail / Marketing Analytics
- Data Source: CSV file

Each row represents one customer and includes:
- Demographic details (Age, Education, Marital Status, Income)
- Product spending (Wine, Meat, Fish, Fruits, Sweets, Baked goods)
- Campaign response information
- Purchase channel activity (Store, Web, Catalog)

---

## 🛠 Data Preparation

The following transformations were performed using Power Query:

- Corrected data types
- Handled missing values in Income column
- Converted date column to proper date format
- Reviewed categorical columns for consistency

No advanced DAX measures were used in this project.

---

## 📈 Dashboard Pages

### 1. Campaign Performance

This page analyzes:
- Sales generated by each campaign
- Campaign response comparison
- Product contribution per campaign
- Channel-wise purchase distribution

Key Observations:
- Campaign 6 generated the highest sales.
- In-store purchases dominate overall.
- Wine is the top-selling product category.

---


### 2. Buyer Composition

This page focuses on customer demographics:
- Average Age: ~55 years
- Average Income: ~$52K
- Majority customers are Graduates
- Most customers are Married
- Most households have no kids or teens

---

### 3. Purchase Drivers

This page explores:
- Income impact on campaign response
- Product spending patterns by age group
- Revenue distribution across income segments

Key Insight:
Higher income customers show stronger campaign acceptance and higher spending.
